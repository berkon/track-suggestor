Track Suggestor

Track Suggestor is a tool which suggests tracks based on the last loaded track in one of Traktor’s decks. It does this based on certain criteria. First a track must roughly match the BPM rate. Then also the key must match either exactly or based on the rules of harmonic mixing.

Track Suggestor connects to Traktor via MIDI. A specific MIDI controller, which is capable of sending track and artist names via MIDI must be configured in Traktor’s Controller Manager. Currently only the Denon DH-HC4500 supports this. Furthermore a virtual MIDI connector must be present on the system. On Apple devices such virtual MIDI ports can be created with onboard tools. On Windows, tools like loopMIDI or CoperLAN can be used.

These are the detailed steps.

First open Traktor, go to Settings and open the Controller Manager. Add a Denon ⇒  DN-HC4500 device. Now set the Out-Port to the virtual MIDI port which you’ve configured on your system. In this case loopMIDI is used on Windows

This means that Traktor will now send track and artist names to this virtual port.

Track suggestor always creates two playlists:

Recommended.nml

This playlist contains tracks roughly matching the BPM rate. Also the key must either match exactly or must be one step up/down or major/minor but then same key.

Recommended_7.nml

This playlist works the same way like the one above, but contains tracks which are 7

steps up/down, which means one half tone. Mixing such tracks still sounds good. In

order to use these tracks the key knob in Traktor must be turned one unit up or down,

which is one half tone up/down and thus matches +/- 7 steps on the key wheel.

Once the MIDI connection is set up, the tool can be started. It will automatically show the settings page at startup as long as setup issues are found. Please configure the following settings:

  • Select the virtual MIDI port to which Traktor was just connected.

  • Select the collection file (collection.nml) which Traktor uses. Don’t worry, it is read only! Nothing will be modified!

  • Select a directory where the playlists containing the suggestions shall be stored to.

IMPORTANT

Traktor does not send track and artist names once to the DN-HC4500 MIDI controller, because the real device is not capable of receiving the data at once. Instead each character is sent one by one and the device is pushing them leftwards in the display thus obtaining the impression that the names are rolling in the display. To obtain this effect characters must not be sent too fast. Thus Track Suggestor has to wait until all characters have been at least sent once. This means that long names can sometimes take 5 (or much more) seconds to load!

IMPORTANT

Traktor isn’t prepared to handle external playlists that change dynamically. Thus when a new track is loaded in Traktor and Track Suggestor in turn creates new playlists based on this new track, these changes are not reflected in the playlist! A workaround must be performed in order to see the changes. First another playlist must be selected (clicked), and then the initially intended one must be selected again.
